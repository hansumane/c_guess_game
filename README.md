# Guess Game in C

## How to use

Compile with *compiler*, *flags* and **make**,
"gcc" is default compiler, and "-Wno-unused-result -std=gnu99 -Os"
are default flags

```
$ make CC=<compiler> FLAG=<flags>
```

Start the game

```
$ ./guess_game
```
